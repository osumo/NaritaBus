技術スタックは以下の通りの内容しか学んでいないので、
その範囲での開発になるよう制限してください。
外部ライブラリを使用しても構いませんが、その際には必ず注意書きと説明を記述してください。

## 開発環境
- **Java**: Version 21
- **ビルドツール**: Maven
- **パッケージング**: Jar
- **IDE**: Eclipse (Pleiades All in One) / Spring Tools
- **OS/CLI**: Windows / Git Bash

## 技術スタック
### **1. Webアプリ開発 (Spring Boot)**
- **フレームワーク**: Spring Boot (MVCモデル)
- **主な学習項目**:
    - Webの基礎（HTTP・ブラウザ・サーバー）
    - コントローラとルーティング
    - リクエストパラメータ・フォーム値の受け渡し
    - Thymeleaf による動的画面生成
    - 入力値のバリデーション
    - DI（依存性注入）
    - セッション管理とログイン機能

### **2. データベース / ORM**
- **RDBMS**: PostgreSQL
- **ORM**: MyBatis (MyBatisによるDBアクセス)
- **主な学習項目**:
    - DBの基礎
    - SQL (SELECT/INSERT/UPDATE/DELETE)
    - DB設計 (ER図、正規化)

### **3. フロントエンド**
- **言語**: HTML / CSS
- **フレームワーク**: Bootstrap
- **手法**: レスポンシブデザイン

### **4. Java 言語詳細**
#### **基礎文法**
- Javaとプログラミングの基礎
- 変数と型
- 演算子
- 条件分岐 / 繰り返し
- 配列
- メソッド
- 複数クラスの連携
- 例外処理
- 標準入力（Scanner）
- 乱数生成（Random）

#### **オブジェクト指向**
- オブジェクト指向の基礎
- コンストラクタ
- 継承
- インターフェース
- カプセル化
- static変数とstaticメソッド

#### **標準ライブラリ**
- コレクション (List, Mapなど)
- 日時操作 (java.time)

### **5. バージョン管理**
- **ツール**: Git
- **主な学習項目**:
    - Git入門（add, commit, push, pullなど）
    - チーム開発（ブランチ管理、Issue活用など）
